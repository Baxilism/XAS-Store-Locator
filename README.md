# 🏪 Xiaomi Authorized Store Locator

An interactive store mapping and location analysis tool for Xiaomi Authorized Stores.

## 📍 Regions

- **[Region 1](./Region%201)** - Jakarta, Bogor, Depok & Bekasi area
  - 21 Active Stores
  - 8 Potential Locations
  - Interactive map with store ratings and customer reviews

## 🗺️ Features

- **Active Store Mapping**: View all operational Xiaomi stores with:
  - Real-time ratings (color-coded by rating)
  - Customer review counts
  - Store addresses
  
- **Potential Location Analysis**: Identify high-opportunity sites for expansion:
  - Strategic street locations
  - Mall partnerships
  - Township developments
  - Strategic insights per location

- **Interactive Map**: Built with Leaflet for smooth navigation and exploration

## 🚀 Quick Start

1. Open the Region files in your browser to view the interactive map
2. Click on store markers to see detailed information
3. Purple diamond markers indicate potential expansion sites

## 📊 Map Legend

- 🟢 **Green dot** - Rating 5.0
- 🟢 **Yellow-green dot** - Rating 4.9
- 🟡 **Gold dot** - Rating 4.8
- 🔴 **Orange dot** - Rating ≤ 4.7
- 🟣 **Purple diamond** - Potential Location
