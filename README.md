## 🗺️ Features

### Active Store Mapping
- Real-time store locations with GPS coordinates
- Customer ratings (5.0 to 4.7 scale)
- Review counts from customer feedback
- Store addresses and contact information
- Color-coded by rating performance

### Potential Location Analysis
- High-opportunity expansion sites
- Strategic analysis by location type:
  - **Street** - Commercial corridors
  - **Mall** - Shopping centers
  - **Township** - Residential developments
- Detailed insights for each potential location

### Interactive Map
- Powered by Leaflet.js for smooth navigation
- Click markers for detailed information
- Zoom and pan to explore areas
- Professional styling with Xiaomi branding

## 💡 How to Use

1. **Visit Homepage**: https://baxilism.github.io/XAS-Store-Locator/
2. **Select Region**: Click "View Map" on any region card
3. **Explore Map**: Click on store markers to see:
   - Store name
   - Customer rating
   - Number of reviews
   - Store address
4. **Identify Opportunities**: Look for purple diamond markers for expansion sites
